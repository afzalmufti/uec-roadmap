Checkerboard concept:
Rearranges the homepage elements in a more dynamic layout, drawing the user down the page, fixed into a chunky tiled grid. Tiles run 3x wide on desktop - could run 2x on mobile, 4x on landscape/large tablet. Use white spaces alongside image tiles to keep the page light and bright.

Contact form

Fairly straight reworking of the contact form on one page, but removed the 123 from desktop (already supressed on live for mobile)

One thing per page - this is the recommended GDS model for simple and complex forms, not literally one thing, but multi-page simple forms perform better than over-grouping of elements on fewer pages.
