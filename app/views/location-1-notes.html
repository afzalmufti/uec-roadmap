Anatomy of a region page

Prototype of a region-level 'location profile' page.

Single large image at the top (Jumbotron)

Because of the dramatic difference between the desktop crop and the mobile, it's probably better to find a different image rather than choose one that works on both.

This jumbotron could in theory work with a few images as a carousel or launch a regional video showcase too.

Intro

No fluff, get straight to the point, 'Why this region is a great place to invest' back it up with facts.

Infographic

Present the intro facts with a graphic, and source link

Full bleed stripe: Quote

This serves as a bit of visual breakup before the map, and a less factual piece of content... if it can link to a relevant story somewhere all the better, but doesn't need to.

Interactive map

Map layers should include Labels and Transport, on by default (probably - test this?). Filters for the industries, one or more should be on by default I think.

The Industry list should have radio buttons (ie. only one at a time to display) as the overload of all the data points isn't really helpful - but it does look pretty, hence the 'show all' button.

From our research, we find that users don't always know how we've 'filed' their particular sector, so the autocomplete field can find and select an industry from the input. This is an idea I'd really like to test with users.

The minimap should link back to a index/intro page for iOPs ... but we are not looking at that design *just* yet.

Full bleed stripe: Image

This should be another break for the page, trying to give it some unique flavour compared to the other regions, trying to connect back to wider GREAT branding and keep that 'full width immersion' feel here.

Regional copy

This is the main area for pulling out more headline attractiveness for this region. Back it up with facts, highlight the fact that provide links to the facts as references below.

Datapoints

This is a simple table with some facts that help understand the region - population, regional GDP/GVA?, cities, industries.

Links

A *small* number of links to the best external resources around. Review this with extreme prejudice - out of date or fluffy sites should not be considered useful.

New footer

This reflects the latest iteration of the ongoing header/nav/footer the rest of the GREAT teams are looking at, so I thought I would include it here. Not added yet is having the translation links in the footer, but they should be.
